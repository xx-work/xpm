# 系统部件管理
> 系统部件管理是针对管理平台的系统部件进行集中管理、识别、监控、处置的功能模块。系统部件包含安全设备, 网络设备, 普通服务器设备，办公设备等。
主要内容包括系统部件的管理策略，进程和监控，变更内容，灾难备份恢复。
